# DevDays 2025 Interaction Kit

這是一個為 DevDays 2025 開發的互動擴充套件管理系統展示專案。本專案展示了即時互動、3D 視覺化以及 WebSocket 通訊的應用。

## 功能特色

- **用戶介面 (User Interface)**: 提供擴充套件的管理與投票功能，支援流暢的動畫與即時互動。
- **星球展示 (Planet Display)**: 3D 視覺化的星球展示系統，根據數據即時改變視覺效果。
- **即時通訊**: 使用 WebSocket 與後端進行低延遲的雙向通訊 (目前連接至 Azure Demo Backend)。

## 技術棧

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend Integration**: WebSocket, REST API (Azure Hosted)
- **Deployment**: GitHub Pages
